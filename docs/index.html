某明星樱花照事件中的谬误史实与虚假信息

[!WARNING|label:荒谬谣言的恶意误导|style:flat]

  • 黛薇夫人的丈夫是印尼第一任总统苏加诺——"中印尼友好的奠基人"1，为什么要篡改资料，张冠李戴成屠华总统苏哈托呢？
  • 靖国神社位于东京市中心，交通便利，占地近10万平方米，怎么会被谣传成“非常偏僻”、“周围没有任何景点”、“仅有一个出入口”？
  • 《刺杀骑士团长》和《海边的卡夫卡》中没有相关内容，为什么凭空捏造其介绍了靖国神社的信息？

   一些不良自媒体，明明可以进行客观理性的科普，加强爱国主义教育，警示日本军国主义对战争罪行的淡化，偏偏要胡编乱造生产谣言，误导舆论博取流量。

   编者无法理解这样的行为。我们的生活不应当被这种荒谬的错误信息充斥，我们赤诚的爱国情绪也不应当被如此恶意引导。 我们有权利了解真实历史和客观事实，拒绝一切恶意误导。也只有如此，才能深入分析和更好地应对我们的“敌人”，真正做到铭记历史，奋发图强。 前事不忘，后事之师。

  • 编者力所能致的行动便是汇总相关资料于此，或许也可称为一种“补课”。原则是有理有据、实事求是地陈述相关事实，纠正误导大众的信息。

  • 所有涉及主观理解的部分会标*和使用突出颜色进行提示，编者仅叙述客观事实，给读者留下自行评判的空间。
