求证 | 是否结交反华、右翼人士？

[!TIP|style:flat|label:Abstract]

  1. 黛薇夫人被换了老公？

    1. 黛薇夫人的丈夫是印尼亲华总统苏加诺，而不是屠华总统苏哈托1。
    2. 2021年10月，黛薇夫人接受采访称自己不反华，并强调丈夫的亲华立场2。
    3. 该明星与黛薇夫人合影为初识，且对日本礼节并不了解3。
    4. *相关百度百科曾遭多次修改。

  2. 鹰羽纹独一无二？由此推断新郎属于极右翼家族？

    1. 科普部分已说明，鹰羽纹为常见服饰租用的通纹。该家纹不独属于右翼家族，平民亦可使用。
    2. 租借新郎、新娘服饰为婚礼流程。日本和国内婚庆公司提供的婚礼流程中都包含租借服饰。
    3. 相关服饰在电商网站上可以随意购买、租赁。

  3. 新郎公司网站首页为靖国神社？

    1. 该公司真正的官网为lenatto.co.jp。
      • 网友向该公司求证，工作人员称"还没建设过中国官网，对于网络传言无可奈何。并且表示如果他们有错误行为，国家一定会依法查处"4。
    2. 网传以"靖国神社"为首页的官网截图存在大量疑点：
      1. 除了最初发布截图的人以外，没有任何人成功进入网站lenatto.com。
      2. 网传以靖国神社为首页的lenatto.com，查询无任何DNS解析记录，应为仅注册域名，而未曾建立网站5,6。
      3. 多处显示公司网址为lenatto.heiit.com，此域名应属于heiit.com，而非lenatto.com7。
      4. 网传截图留错公司电话，拼错公司名字4。
    3. *平心而论，
      1. 一家化妆品公司，在日本官网放产品相关的内容，在中国官网放靖国神社和鸟居？
      2. 如果该公司官网真为靖国神社图片，必然会面临道歉、下架、退出中国市场。为何在舆情沸腾、大量举报之后，还能在国内正常经营？

  4. 右翼婚礼？反华分子元谷外志雄？右翼俱乐部？

    1. 对比照片可知，网传截图中的宾客并非元谷外志雄。
    2. 该宾客被否认为元谷后，谣言又改称此人为右翼俱乐部VAV的会长。
    3. VAV俱乐部是商业性质的交流会，入会无门槛。该俱乐部非右翼背景，相反，其有诸多左翼会员和亲华派人士。

  5. 与小泉孝太郎亲密合影？

    1. 对比照片可知，合影的朋友与小泉纯一郎之子小泉孝太郎，明显不是同一人。
    2. 实属空穴来风的谣言。
